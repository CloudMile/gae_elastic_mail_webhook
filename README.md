# WebHook API Server for ElasticMail

```
$ git clone git@github.com:CloudMile/gae_elastic_mail_webhook.git
```

## Deploy
Edit `app.yaml`, change `<gcp_project_id>` to your GCP/GAE project id
```
$ gcloud app deploy app.yaml
```
